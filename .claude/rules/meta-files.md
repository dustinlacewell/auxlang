# Meta Files

Location: `lang-research/meta/`

## Formats

- **decisions-made.md** - `| ID | Decision | Date | Rationale |` - one line per decision
- **worklog.md** - Bullets under date headers, one line per item
- **current-task.md** - Brief: what's done, what's next
- **context-letter.md** - Quick orientation, points to docs

## Docs

Stable knowledge lives in `docs/`:
- `audio-model.md`, `sequencer.md`, `descriptors.md`, `live-reeval.md`, `wasm-devices.md`

## Plans & Archive

- `plans/` - Future work only
- `archive/` - Completed plans and early research

## When to Update

- **decisions-made.md**: Architectural choices
- **worklog.md**: After completing work
- **current-task.md**: When focus changes
- **context-letter.md**: When major features complete
